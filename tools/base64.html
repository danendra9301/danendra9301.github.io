Base64 Encoder / Decoder

Encode teks/UTF-8 ke Base64 atau decode Base64 ke teks. Aman untuk karakter Unicode (UTF-8 byte-safe).

Tentang Tool Ini

Base64 adalah skema encoding yang mengubah data binary menjadi string ASCII (huruf, angka, plus dan slash). Sering dipakai untuk embed gambar di CSS (data URI), encode email attachment, atau passing data binary lewat protokol text-only seperti JSON/XML. Tool ini support UTF-8 byte-safe - bisa encode emoji, karakter Mandarin, Arab, dan lainnya tanpa rusak.

Cara Kerja

Pilih mode Encode (text ke Base64) atau Decode (Base64 ke text). Ketik di input, hasil real-time di output. Tombol Swap untuk tukar input/output dengan cepat (untuk round-trip testing).

Kapan Memakai Tool Ini

  • Encode gambar kecil jadi data URI untuk inline di HTML/CSS
  • Decode token JWT untuk inspect payload
  • Encode credential atau data binary untuk passing via URL
  • Belajar konsep encoding untuk security/development

Pertanyaan yang Sering Diajukan

Apa bedanya Base64 dengan enkripsi?

Base64 BUKAN enkripsi - hanya encoding. Siapapun bisa decode. Untuk security, butuh enkripsi seperti AES.

Kenapa Base64 lebih besar dari aslinya?

Setiap 3 byte original jadi 4 karakter Base64. Overhead sekitar 33%. Itu trade-off untuk membuat data binary aman dilewatkan via text channel.

Bisa decode binary file (gambar, dll)?

Tool ini fokus text. Untuk decode Base64 jadi file binary, butuh tool/script terpisah.

Aman pakai untuk password?

TIDAK. Base64 mudah di-decode. Untuk password storage, pakai hashing (bcrypt, argon2).
