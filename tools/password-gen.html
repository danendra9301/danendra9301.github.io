Tentang Tool Ini

Password kuat adalah pertahanan pertama akun online dari serangan brute-force dan credential stuffing. Tool ini generate password acak pakai crypto.getRandomValues - sumber acak kriptografis browser yang dipakai juga oleh library crypto pro. Custom panjang (6-128 karakter), pilih karakter set (huruf besar/kecil/angka/simbol), dan opsi hindari karakter ambigu (0Oo1Il).

Cara Kerja

Atur opsi karakter (default semua kategori aktif), geser slider panjang (minimum 16 untuk akun penting), tool generate password baru. Strength meter menampilkan estimasi entropy dalam bits.

Kapan Memakai Tool Ini

  • Membuat password untuk akun baru (email, sosmed, banking)
  • Update password lama yang lemah
  • Generate API key atau token untuk aplikasi
  • Membuat passphrase untuk encryption key

Pertanyaan yang Sering Diajukan

Berapa panjang password ideal?

Minimum 12 karakter untuk akun biasa, 16+ untuk email/banking, 24+ untuk master password (password manager).

Apa itu entropy bits?

Ukuran ketidakpastian - makin tinggi makin susah ditebak. 60+ bits sama dengan strong, 80+ very strong, 100+ paranoid level.

crypto.getRandomValues itu aman?

Ya, ini Web Crypto API yang dipakai semua browser modern. Acaknya kriptografis, bukan Math.random yang predictable.

Apakah password dikirim ke server?

Tidak sama sekali. Semua generate di browser Anda. Tutup tab sama dengan password hilang dari memori.

Privasi dan Keamanan

ConvertHUB dibangun dengan prinsip privacy-first: semua proses berjalan 100% di browser Anda menggunakan teknologi modern seperti Canvas API, WebAssembly, dan Web Crypto. File yang Anda upload tidak pernah dikirim ke server kami atau pihak ketiga. Begitu Anda tutup tab, semua data hilang dari memori. Tidak ada akun, tidak ada tracking konten file, tidak ada database. Anda punya kontrol penuh atas data Anda - cocok untuk file sensitif seperti dokumen pribadi, password, atau konten yang belum dipublikasikan.
