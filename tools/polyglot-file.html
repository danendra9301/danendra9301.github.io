Apa itu Polyglot File?

Polyglot file adalah satu berkas tunggal yang valid dalam lebih dari satu format sekaligus. Contoh: file yang sama bisa dibuka sebagai gambar PNG dan diekstrak sebagai arsip ZIP — tergantung ekstensi yang dipakai saat membukanya.

1 File "Wajah Depan" (Wrapper)

Format ini yang akan terlihat saat file dibuka biasa. Dukungan: PNG, JPG, GIF, BMP, WebP, PDF.

2 File yang Disembunyikan (di dalam ZIP)

File-file ini akan tersembunyi di dalam polyglot, dan bisa diekstrak kalau file di-rename jadi .zip. Boleh upload banyak file.
